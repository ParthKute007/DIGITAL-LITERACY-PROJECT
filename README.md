##  Module Summaries

### Task 1 — Digital Literacy Infographic (Module 1)
Created a one-page infographic using **Canva** covering what digital literacy is, safe internet practices, and professional online presence. The design uses icons and colour-coded sections to make the content visually accessible for first-year students.

### Task 2 — Student Digital Portfolio (Module 2)
Set up professional profiles on **GitHub**, **LinkedIn**, and **Kaggle**. Each platform serves a distinct purpose — GitHub for code projects, LinkedIn for professional networking, and Kaggle for data science practice.

### Task 3 — Coding & Collaboration Platforms (Module 3)
Completed a beginner challenge on **HackerRank** and built a 5-question **Digital Literacy Awareness Quiz** using Google Forms, linked to a Google Sheet for response tracking.

### Task 4 — Email Etiquette (Module 4)
Drafted two professional emails (deadline extension request + internship interest) and created a Social Media Do's and Don'ts checklist with 5 do's and 5 don'ts for college students.

### Task 5 — Cybercrime Awareness (Module 5)
Researched **UPI/Online Payment Fraud**, wrote a 200-word case study, and created a prevention checklist with 8+ actionable tips tailored to Indian college students, including UPI-specific safety and the Cyber Crime Helpline 1930.
