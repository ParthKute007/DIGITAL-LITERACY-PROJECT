## About This Project

This repository is my Digital Literacy Portfolio, created as part of the CSE0001 course project at VIT Bhopal University. As a Student Digital Ambassador, the goal of this project is to build essential digital skills — from creating professional online profiles, to understanding email communication, coding platforms, and cybercrime awareness.

All five tasks are documented here, along with the final project report.

## Module Summaries

### Task 1 – Digital Literacy Infographic (Module 1)
Created a one-page infographic using Canva covering what digital literacy is, safe internet practices, and professional online presence. The design was made to be simple and easy to understand for first-year students.

### Task 2 – Student Digital Portfolio (Module 2)
Set up profiles on GitHub, LinkedIn, and Kaggle. Created a GitHub profile README and updated the LinkedIn Education section with current degree details.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed a beginner-level challenge on HackerRank and created a 5-question Digital Literacy Awareness Quiz using Google Forms.

🔗 **Google Form Quiz Link:** *(Add your Google Form link here after creating it)*

### Task 4 – Professional Email & Etiquette (Module 4)
Drafted two professional emails — one requesting a deadline extension and one expressing interest in an internship. Also created a Social Media Do's and Don'ts checklist.

### Task 5 – Cybercrime Awareness (Module 5)
Researched and wrote a case study on Phishing attacks, including how they happen, who is targeted, and what the consequences are. Created a 'Stay Safe Online' checklist with actionable tips for college students.
